## Contributing

Thanks for contributing!

### Development

- Node.js 18+
- Copy config: `cp .env.example .env.local`
- Start proxies: `npm run fast` and/or `npm run claude`
- Run smoke checks: `npm run smoke:all`

### Pull Requests

- Keep changes focused and minimal.
- Update docs when behavior/config changes.
- Do not commit secrets (API keys, tokens, `.env*` files).

### Code Style

- Prefer small, readable functions and minimal dependencies.
- Follow existing formatting in the repo.
